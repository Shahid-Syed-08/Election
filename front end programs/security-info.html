Security Best Practices for Users

Do's

  • Use strong, unique passwords for each account
  • Enable two-factor authentication when available
  • Keep your software and browsers updated
  • Log out when using shared computers
  • Report suspicious activities immediately

Don'ts

  • Share your login credentials with anyone
  • Use weak passwords like "123456" or "password"
  • Click on suspicious links or attachments
  • Use public Wi-Fi for sensitive transactions
  • Ignore security warnings from your browser

Quick Security Tips

Password Management

Use a password manager to generate and store strong, unique passwords for all your accounts.

Mobile Security

Keep your mobile devices updated and use biometric authentication when possible.

Network Security

Use a VPN when connecting to public Wi-Fi networks to encrypt your data transmission.
